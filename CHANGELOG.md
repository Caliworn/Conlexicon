# Changelog

本文件记录 Conlexicon 的重要功能、行为与兼容性变更。

## New

### 新增

- 质量问题高级筛选会在词条列表卡片下方显示对应的问题类型小卡片，并在悬浮时展示包含优先度、模块和具体问题的详情卡。
- 词条列表高级筛选条新增刷新按钮，并将循环切换按钮独立为刷新按钮右侧的单独控件。
- 词条列表高级筛选控制统一为单行控制条，长筛选名称会优先截断以保留刷新、循环和退出按钮。
- 数据分析质量检查页将类别切换与高级筛选入口整合：类别按钮保留数量与空状态变灰，只切换本页问题列表；“在词条列表查看”按钮负责进入高级筛选。
- 词条列表的基础词性筛选新增“无词性”选项，可筛出未识别到词性的词条。
- 数据分析中的词性分布，以及标签频率中被当前词典识别为词性的标签，现在会触发词条列表的基础词性筛选，而不是标签高级筛选。
- 质量检查新增 IPA 问题类别，纳入缺少 IPA 与多个主重音检查；多个主重音按中优先级处理，并参与模块循环筛选。

### 修复

- 修复高级筛选刷新后当前筛选已无匹配对象时仍残留旧词条的问题；若其他可循环变体仍有结果，则继续显示循环按钮。
- 修复无标签或无释义词条在词条列表卡片中标签与质量问题小卡片位置、尺寸异常。
- 修复缺少释义/搜索片段的词条卡片仍保留普通卡片最小高度，导致标签或质量问题标签上方出现大块空白的问题。
- 修复质量问题高级筛选在切换界面语言后，当前筛选名称和词条卡片问题标签仍停留在旧语言的问题。
- 修复标签组合、词性、词长、IPA 音位等统计型高级筛选名称在切换界面语言后未完整更新的问题。
- 修复 IPA 自动生成一致、宽松不一致、严格不一致等循环高级筛选名称在切换界面语言后未更新的问题。
- 修复词条列表高级筛选控制条中标签、图标按钮和退出按钮在中英文模式下实际高度与垂直对齐不一致的问题。
- 修复词条列表高级筛选标签过长被省略时无法查看完整内容的问题；溢出标签现在会使用应用内 tooltip 显示完整名称。
- 修复质量问题高级筛选中词条卡片位于列表底部时，质量问题悬浮详情向下展开被截断的问题；下方空间不足时会自动改为向上展开。
- 修复向上展开的质量问题悬浮详情在鼠标移开淡出时短暂跳回下方的问题。
- 修复质量问题悬浮详情在列表底部附近触发时可能扩大滚动区域、导致滚动条闪烁且详情卡无法稳定显示的问题。
- 修复宽屏英文界面下 Conlexicon 标题与词典管理、导航收起按钮重叠的问题；展开态工具导航会保持按钮同一行，并响应式缩小英文标题字号。

## 2026-06-30

### 新增

- 其他设置新增词性标签配置；可手动指定哪些词条标签识别为词性，并支持同一词条在词条列表、详情、筛选和数据分析中显示与统计多个词性。
- 其他设置新增自动整理标签顺序，可按统一标签列表批量重排词条标签并在执行前弹窗确认。
- 数据分析的 IPA 自动生成检查新增“一致”统计项，并可与宽松不一致、严格不一致高级筛选互相切换。
- 数据分析覆盖率条目的高级筛选入口统一为可循环筛选；有释义、IPA、形态表格与总览入口保持一致，有例句、有备注、有来源可切换到对应的“无”筛选。
- 数据分析质量检查页的选项卡和高级筛选入口按优先度、检查模块分组；模块筛选覆盖词形、标签、词源网络、Glossed 例句和其他问题，并支持循环切换。

### 改进

- 词典管理、其他设置、自动 IPA、自动形态学、语言文档和语料库的未保存离开确认改为“保存 / 放弃更改 / 取消”三选项；语言文档和语料库在自动保存关闭时适用。
- 高级筛选循环切换按钮的箭头图标改为居中显示。

## 2026-06-24

### 新增

- 新增词典级语料库，支持有序语料块、发言人/模态语料层、独立语料单元、父级链接、属性继承、标签与备注。
- 新增语料完整性检查，检测缺失单元、重复父级与实体 ID 冲突。
- 新增语料单元 Gloss 名称渲染，并可分别配置单元卡片、单元内容标题和词条例句的渲染对象与对齐方式。
- 新增 `\gla`、`\glb`、`\glc`、`\ft` 的独立字体、字号、粗体、斜体与 small caps 样式设置。
- 新增语料库与语言文档自动保存设置，并在关闭页面或程序时执行最终保存。
- 新增词典导入 ID 冲突确认，可选择覆盖现有词典或取消导入。
- 新增全局界面语言记忆，保存在 `data/index.json` 中。
- 新增词条列表和词条详情标签点击筛选；主标签使用普通词性筛选，次级标签使用高级筛选。
- 新增词典级左侧工具导航排序。
- 自动 IPA 映射规则支持拖动卡片调整顺序，并按词典保存规则优先级。
- 桌面工具导航支持在完整导航与图标栏之间切换；宽屏默认展开并在当前会话记忆选择，中等宽度强制使用图标栏且隐藏切换按钮。图标栏提供稳定的中英文悬浮/焦点提示、无障碍状态和更直观的功能图标。
- 桌面词条列表支持完全收起与重新展开，纯图标按钮置于词典标题之后，使用应用内提示以及展开向下、收起向右的箭头；主内容区会自动扩宽，搜索、排序和虚拟列表状态在切换后保持不变，并修复 800–980px 双栏布局下列表高度过早受限的问题。

### 改进

- 页面切换时的局部编辑和完整编辑行为改为“保存”“放弃更改”或“弹窗提示”三种模式。
- 词根模式跳转到未展开的衍生词时会自动展开目标词根，并正确定位多来源词条的对应实例。
- 语料单元编辑内容现在可以即时更新渲染预览，不必等待保存完成。
- 自动保存不再退出当前语料编辑状态，也不会清除当前草稿和撤销历史。
- 数据分析与其他设置卡片改为响应式瀑布流布局，长卡片不再撑开同一网格行。
- 自动 IPA、自动形态学和其他设置移除不必要的最外层大卡片，使子区域边界更清晰。
- 其他设置改进宽屏空间利用，并调整渲染对象、标签替换等长文本输入区域。
- README 补充语料库、Gloss 样式、保存策略、导入保护与全局语言设置说明。
- IPA 映射改为严格按照列表从上到下匹配；规则始终读取原始词形，生成结果不会被后续规则再次读取，较早规则消耗的字符不会再参与后续匹配。
- 改进中间宽度与窄屏下的通用布局收缩和换行；词条详情、设置、分析等卡片不再被内部 Grid、Flex 或长文本撑出边界。
- 形态表格与对齐 Gloss 在空间不足时保持卡片内部局部横向滚动，不再引起页面级横向滚动。

### 性能

- 全局渲染改为只更新当前打开的功能模块；未打开数据分析时不再生成分析报告。
- 词条列表、词根模式衍生词和语料列表加入动态高度虚拟滚动，页面只保留可见区域附近的卡片。
- 虚拟列表保留选中项跳转、词根展开、多来源上下文与窄屏独立滚动行为。

### 修复

- 修复语料库为空时仍存在额外滚动空间的问题。
- 修复 Gloss 多行对齐未生效、`\ft` 参与对齐以及对齐后字号异常缩小的问题。
- 修复语料自动保存导致编辑状态、草稿和撤销记录丢失的问题。
- 修复词根模式下跳转到衍生词时可能定位到错误实例的问题。
- 修复设置和分析网格因超长卡片产生大面积垂直空白的问题。
- 修复原生单行输入框裁切 `d͡ʒ`、`t͡ʃ` 等跨字符组合附加符号的问题；IPA 映射、音节规则、沙盒输入以及完整和局部词条发音现在使用保持单行行为的文本区域。
- 修复特定窗口比例下词条详情子卡片和操作区超出父卡片，以及最窄视口仍出现页面横向滚动的问题。

### 安全与兼容性

- 词条、语料块、语料层和语料单元使用跨类型实体 ID 唯一性检查，前后端保存时都会阻止重复 ID。
- 保持旧词典 JSON 的配置默认值与数据结构迁移逻辑，新增设置缺失时会使用兼容默认值。
- `data/` 继续保持 Git 忽略，个人词典不会进入版本控制。
