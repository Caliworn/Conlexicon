# Changelog

本文件记录 Conlexicon 的重要功能、行为与兼容性变更。

## Next

### 新增

- 自动 IPA 映射规则支持拖动卡片调整顺序，并按词典保存规则优先级。

### 改进

- IPA 映射改为严格按照列表从上到下匹配；规则始终读取原始词形，生成结果不会被后续规则再次读取，较早规则消耗的字符不会再参与后续匹配。
- 改进中间宽度与窄屏下的通用布局收缩和换行；词条详情、设置、分析等卡片不再被内部 Grid、Flex 或长文本撑出边界。
- 形态表格与对齐 Gloss 在空间不足时保持卡片内部局部横向滚动，不再引起页面级横向滚动。

### 修复

- 修复原生单行输入框裁切 `d͡ʒ`、`t͡ʃ` 等跨字符组合附加符号的问题；IPA 映射、音节规则、沙盒输入以及完整和局部词条发音现在使用保持单行行为的文本区域。
- 修复特定窗口比例下词条详情子卡片和操作区超出父卡片，以及最窄视口仍出现页面横向滚动的问题。

## 2026-06-24

### 新增

- 新增词典级语料库，支持有序语料块、发言人/模态语料层、独立语料单元、父级链接、属性继承、标签与备注。
- 新增语料完整性检查，检测缺失单元、重复父级与实体 ID 冲突。
- 新增语料单元 Gloss 名称渲染，并可分别配置单元卡片、单元内容标题和词条例句的渲染对象与对齐方式。
- 新增 `\gla`、`\glb`、`\glc`、`\ft` 的独立字体、字号、粗体、斜体与 small caps 样式设置。
- 新增语料库与语言文档自动保存设置，并在关闭页面或程序时执行最终保存。
- 新增词典导入 ID 冲突确认，可选择覆盖现有词典或取消导入。
- 新增全局界面语言记忆，保存在 `data/index.json` 中。
- 新增词条浏览栏和词条详情标签点击筛选；主标签使用普通词性筛选，次级标签使用高级筛选。
- 新增词典级左侧工具导航排序。

### 改进

- 页面切换时的局部编辑和完整编辑行为改为“保存”“放弃更改”或“弹窗提示”三种模式。
- 词根模式跳转到未展开的衍生词时会自动展开目标词根，并正确定位多来源词条的对应实例。
- 语料单元编辑内容现在可以即时更新渲染预览，不必等待保存完成。
- 自动保存不再退出当前语料编辑状态，也不会清除当前草稿和撤销历史。
- 数据分析与其他设置卡片改为响应式瀑布流布局，长卡片不再撑开同一网格行。
- 自动 IPA、自动形态学和其他设置移除不必要的最外层大卡片，使子区域边界更清晰。
- 其他设置改进宽屏空间利用，并调整渲染对象、标签替换等长文本输入区域。
- README 补充语料库、Gloss 样式、保存策略、导入保护与全局语言设置说明。

### 性能

- 全局渲染改为只更新当前打开的功能模块；未打开数据分析时不再生成分析报告。
- 词条列表、词根模式衍生词和语料列表加入动态高度虚拟滚动，页面只保留可见区域附近的卡片。
- 虚拟列表保留选中项跳转、词根展开、多来源上下文与窄屏独立滚动行为。

### 修复

- 修复语料库为空时仍存在额外滚动空间的问题。
- 修复 Gloss 多行对齐未生效、`\ft` 参与对齐以及对齐后字号异常缩小的问题。
- 修复语料自动保存导致编辑状态、草稿和撤销记录丢失的问题。
- 修复词根模式下跳转到衍生词时可能定位到错误实例的问题。
- 修复设置和分析网格因超长卡片产生大面积垂直空白的问题。

### 安全与兼容性

- 词条、语料块、语料层和语料单元使用跨类型实体 ID 唯一性检查，前后端保存时都会阻止重复 ID。
- 保持旧词典 JSON 的配置默认值与数据结构迁移逻辑，新增设置缺失时会使用兼容默认值。
- `data/` 继续保持 Git 忽略，个人词典不会进入版本控制。
